(
  • None
)
Void

Defined in subnav-new.js:135

Sets underlines for subnav links based off helpx logic If the current page is part of the Hub Family, then the 'Learn & Support' link is underlined If the current page is part of the Beginner's Guide Family, then the 'Beginner's Guide' link is underlined If the current page is part of the Learn Family, then the 'Tutorials' link is underlined If the current page is part of the User Guide Family, then the 'user Guide' link is underlined
